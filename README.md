# Neon Blocks

A visually striking, fully-featured pure HTML5/JS/CSS Breakout clone. 
Playable directly in your browser with zero build steps or backend servers required!

## Features
- **100 Procedurally Generated Stages**: Face increasingly complex patterns including stripes, checkerboards, pyramids, and chaotic noise blocks!
- **Arcade Power-Ups**: Breaking bricks gives you a chance to collect neon drops:
  - `< >` **(Cyan)**: Expand your paddle for an easier catch!
  - `1UP` **(Pink)**: Gain an extra life.
  - `x3` **(Green)**: Multi-ball frenzy.
  - `SLW` **(Yellow)**: Slows down all balls.
  - `GUN` **(Red)**: Blast bricks with dual-mounted lasers!
- **Synthesizer Sound Engine**: Built entirely on the Web Audio API for authentic, procedurally generated arcade bings, bops, and laser squeals.
- **Cinematic Ending**: Beat all 100 stages to see the animated staff scroll credits!
- **Cross-Platform Controls**:
  - **Desktop**: Use your mouse or Left/Right Arrow keys. Press Spacebar to shoot lasers and launch the ball.
  - **Mobile**: Full touch support with an on-screen Shoot button.

## How to Play
1. Download or clone this repository.
2. Double-click `index.html` to open it in any modern web browser.
3. Turn up your volume.
4. Hit PLAY!
